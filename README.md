# 📊 CareerSync | AI-Powered Student Placement Analytics Portal

CareerSync is a full-stack, data-driven career readiness analytics workspace designed to evaluate student profiles, predict placement probabilities, and map clear technical learning trajectories. 

Live Application: **[https://careersyncsite.streamlit.app/]**

---

## 🚀 Key Features

*   **Secured Student Workspace**: Built-in credential validation gate syncing authenticated accounts straight into dedicated cloud schemas.
*   **Localized Cloud Matrix**: Fully integrated with a high-performance database instance running on a South Asia cluster to ensure ultra-low latency queries.
*   **Dynamic "What-If" Analysis Engine**: Real-time simulation testing letting students visually track how adding specific technical competencies or certs directly impacts their statistical readiness metrics.
*   **ATS Keyword Verification**: Deep architectural parsing mapping core target deficiencies, keyword congruence, and metric tracking.
*   **Interactive Preparation Checkpoints**: Consolidated framework tracker featuring interactive trial challenge prompts and downloadable mock prep data logs.

---

## 🛠️ The Tech Stack

*   **Frontend Interface:** Streamlit (Custom premium glassmorphism theme layout)
*   **Database Cloud Architecture:** Supabase (PostgreSQL engine)
*   **Analytical Computations:** Python 3.11 (Pandas, NumPy, Plotly Graph Objects)
*   **Source Version Control:** Git (Fully managed local and remote repositories)
